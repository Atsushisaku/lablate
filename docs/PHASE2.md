# Lablate - Phase 2 仕様書

## 目標

グラフ・表ブロックを追加し、PPT代替としての価値を示す。
フェーズ1のUIデモへのフィードバックを反映した上で着手する。

---

## 実装方針（決定事項）

### テーブルブロック：Jspreadsheet CE を採用

スプレッドシートUIの自作は複雑性が高く（セル参照、数式エンジン、キーボードナビゲーション、範囲選択など）、
公開ライブラリ **Jspreadsheet Community Edition（MIT ライセンス）** を採用する。

| 項目 | 内容 |
|---|---|
| ライブラリ | [jspreadsheet/ce](https://github.com/jspreadsheet/ce) |
| ライセンス | MIT |
| 数式 | Excel 互換（組み込み済み） |
| UI | Google Sheets / Excel ライク（キーボードナビ・範囲選択・コピペ対応） |
| 統合方法 | BlockNote カスタムブロックとして `useRef + useEffect` でDOMにマウント |

**自作からの移行理由：**
- flushSync・stale closure・正規表現など実装上の問題が継続して発生
- スプレッドシートは「一見シンプルだが実装難易度が高いUI」の代表例
- Jspreadsheet CE で同等以上の機能が即座に利用可能

## テーブルブロック（CsvTableBlock）

### 基本機能

- セル編集・行・列の追加・削除
- CSV インポート（ファイル選択 UI）
- 列ヘッダーは常に A, B, C… / 行インデックスは 1, 2, 3…（Excel 互換）
- CSV 読み込み時はヘッダー行も含めすべてセルデータとして扱う

### 数式（Excel 互換）

- セル参照：`=A1+B2`
- 範囲関数：`=SUM(A1:A5)`、`=AVERAGE(A1:A5)`、`=MAX(A1:A5)` など
- 四則演算・ネスト・複数セル参照
- 数式入力中に矢印キーでセル参照を挿入可能（Google Sheets 風）
  - `=` 入力後に矢印キーでセル参照が自動挿入される
  - 続けて矢印を押すと参照先が更新される
  - 演算子（+, -, *, / 等）を入力すると参照が確定し、次の矢印で新しい参照を開始
  - 参照中のセルは紫色の破線枠でハイライト表示

### Google Sheets ライクなキーボード操作

ProseMirror（BlockNote）がキーイベントを横取りするため、document capture フェーズでキーボードイベントをインターセプトして Google Sheets ライクな操作性を実現。

| 操作 | 動作 |
|---|---|
| 矢印キー | セル移動（Shift で範囲選択、Ctrl で端まで移動） |
| Enter | 編集モードに入る |
| Enter（編集中） | 確定して下のセルに移動 |
| F2 | 編集モードに入る |
| Escape（編集中） | 編集をキャンセル |
| Tab / Shift+Tab | 右 / 左に移動 |
| Delete / Backspace | 選択セルの内容をクリア |
| 文字入力 | セル内容を置き換えて編集開始 |
| Ctrl+Z / Ctrl+Y | Undo / Redo |
| Ctrl+C / Ctrl+X | コピー / カット |
| Ctrl+V | クリップボードからペースト |
| Ctrl+A | 全セル選択 |

### UI コンポーネント

**ツールバー（展開時）：**
- 最小化 / 展開ボタン
- CSV 読み込みボタン
- グラフ追加ボタン → ChartBlock を直下に挿入
- タイトル表示トグルボタン
- 行追加（+行）ボタン
- 列追加（+列）ボタン

**タイトル：**
- 「タイトル」ボタンで表示 / 非表示を切り替え
- 折りたたみ時はタイトルと行列数（例: "3 列 × 5 行"）を表示

**行・列の削除：**
- 行インデックスセル・列ヘッダーにホバーするとゴミ箱アイコンが表示
- クリックで該当行 / 列を削除

**セル選択の表示：**
- 現在セル：青色（#1a73e8）4辺ソリッド枠線、塗りつぶしなし
- 範囲選択：青色4辺ソリッド枠線 + 薄い青背景
- 行インデックス列：最小幅（32px）

### データ永続化

- テーブルデータは `lablate_dataset_<datasetId>` として localStorage に保存
- データ変更時に `CustomEvent("lablate-dataset-change")` を発火し、グラフブロックに通知

## グラフブロック（ChartBlock）

### 基本機能

- テーブルデータから XY 散布図を生成（Plotly.js）
- テーブルのデータ変更をリアルタイムに反映（CustomEvent による連携）
- 最小化 / 展開対応（折りたたみ時はタイトルまたは軸名を表示）

### 設定項目

| 設定 | 説明 |
|---|---|
| グラフタイトル | テキスト入力（空の場合は "Y vs X" 形式で自動生成） |
| X 軸列 | ドロップダウンで選択（共通） |
| Y 軸列 | 複数選択可能（タグ UI で追加 / 削除） |
| ヘッダー行 | 列ラベルとして使用する行（なし / 行番号） |
| データ行範囲 | 開始行〜終了行（空 = 全行） |
| 凡例 | 表示 / 非表示トグル（グラフ内右側に表示） |

### 軸範囲の変更

- Plotly のビルトインのドラッグズームで軸範囲を変更可能
- プロット領域の四辺に薄いグレー（#d1d5db）の境界線を表示し、操作可能エリアを示す
- ホバー時にモードバー（ズーム / パン / リセット等）を表示

### リサイズ機能

- ブロックをクリックすると8つのリサイズハンドル（角4つ + 辺中央4つ）が表示
- ドラッグでブロック全体の幅・高さを変更
- ドラッグ中は青い破線枠とサイズ表示（"W × H"）のオーバーレイ
- サイズは `chartWidth`（0 = 自動）・`chartHeight`（デフォルト 340px）として設定に保存
- Plotly は `responsive: true` + `autosize: true` でコンテナに追従

### ブロック構造

```typescript
createReactBlockSpec({
  type: "chart",
  propSchema: { datasetId: { default: "" } },
  content: "none",
})
```

## グラフ機能拡張（Phase 2.1）

### データ範囲設定の折りたたみ

グラフブロックのツールバーを **常時表示部分** と **詳細設定（デフォルト非表示）** に分離する。

**常時表示：**
- グラフタイトル入力
- X 軸列・Y 軸列セレクタ
- 回帰式トグル・次数選択（後述）
- 「詳細 ▶」トグルボタン

**詳細設定（折りたたみ、デフォルト非表示）：**
- ヘッダー行指定
- データ行範囲（開始行〜終了行）

折りたたみ状態はグラフ設定（`lablate_chart_config_<blockId>`）に `showDataRange: boolean` として永続化する。

---

### 複数系列プロット

1つのX軸に対して複数のY列を重ねてプロットできる。

- Y列はタグUI `[B 📈 ×] [C ×] [+▼]` で追加・削除
- 8色のカラーパレット（blue, red, green, amber, purple, pink, cyan, orange）で系列を自動色分け
- 凡例はグラフ内右側に表示（表示 / 非表示トグル）

---

### 多項式回帰とR²表示

データ点に対して **1〜3次の多項式回帰** を計算し、回帰線・式・決定係数をグラフ上に表示する。

**計算方法：**
- 最小二乗法（正規方程式 / Vandermonde 行列）による多項式フィッティング
- 外部ライブラリ不要（自前実装）

**表示：**
- Plotly の line trace として回帰曲線を描画（**系列と同色の破線**、線幅2）
- グラフ左上に annotation で回帰式と R² を系列色で表示
  - 例：`B: y = 1.23x² + 0.45x − 2.10  (R² = 0.974)`
- 有効数字は適切に丸める（係数4桁、R²は小数点以下3桁）

**UI：**
- 各Y列タグ内にトレンドラインアイコン（SVG曲線）で系列ごとに回帰ON/OFF
- 次数セレクタ（1次 / 2次 / 3次）は全系列共通
- ON時: アイコンが系列色の破線表示 / OFF時: 薄いグレー

**ChartConfig：**

```typescript
export type ChartConfig = {
  xColumn: string;
  yColumns: string[];            // 複数Y列
  title: string;
  headerRow: string;
  dataStartRow: string;
  dataEndRow: string;
  showDataRange: boolean;        // デフォルト: false
  showLegend: boolean;           // デフォルト: true
  regressionDegree: number;      // 1 | 2 | 3, デフォルト: 1（全系列共通）
  regressionColumns: string[];   // 回帰を表示するY列インデックス
};
```

### スプレッドシートタブからのグラフ挿入

「md に挿入」ボタンでスプレッドシートタブのグラフを md 内に挿入する際、
グラフの全設定（X/Y列、ヘッダー行、回帰線、凡例等）が反映される。

---

### テーブルタイトルの統一

テーブルのタイトルは `DatasetMeta.name`（データセットレジストリ）に一本化。
- md 内のテーブルタイトル変更 → タブ名に反映
- スプレッドシートタブ内ではタイトル入力欄を非表示（タブ名がタイトル代わり）
- `lablate-dataset-rename` カスタムイベントで同期

---

### ページリンクブロック（PageLinkBlock）

Notion 風のページリンクブロック。`/ページリンク` スラッシュメニューから任意の位置に挿入。

- `📄 ページタイトル` 形式で表示
- クリックで対象ページに遷移（`lablate-navigate-page` カスタムイベント経由）
- ページタイトル変更時にリンク表示名が自動更新（`lablate-tree-change` イベント）

```typescript
createReactBlockSpec({
  type: "pageLink",
  propSchema: { pageId: { default: "" } },
  content: "none",
})
```
