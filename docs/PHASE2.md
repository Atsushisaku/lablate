# Lablate - Phase 2 仕様書

## 目標

グラフ・表ブロックを追加し、PPT代替としての価値を示す。
フェーズ1のUIデモへのフィードバックを反映した上で着手する。

---

## 実装方針（決定事項）

### テーブルブロック：Jspreadsheet CE を採用

スプレッドシートUIの自作は複雑性が高く（セル参照、数式エンジン、キーボードナビゲーション、範囲選択など）、
公開ライブラリ **Jspreadsheet Community Edition（MIT ライセンス）** を採用する。

| 項目 | 内容 |
|---|---|
| ライブラリ | [jspreadsheet/ce](https://github.com/jspreadsheet/ce) |
| ライセンス | MIT |
| 数式 | Excel 互換（組み込み済み） |
| UI | Google Sheets / Excel ライク（キーボードナビ・範囲選択・コピペ対応） |
| 統合方法 | BlockNote カスタムブロックとして `useRef + useEffect` でDOMにマウント |

**自作からの移行理由：**
- flushSync・stale closure・正規表現など実装上の問題が継続して発生
- スプレッドシートは「一見シンプルだが実装難易度が高いUI」の代表例
- Jspreadsheet CE で同等以上の機能が即座に利用可能

## テーブルブロック（CsvTableBlock）

### 基本機能

- セル編集・行・列の追加・削除
- CSV インポート（ファイル選択 UI）
- 列ヘッダーは常に A, B, C… / 行インデックスは 1, 2, 3…（Excel 互換）
- CSV 読み込み時はヘッダー行も含めすべてセルデータとして扱う

### 数式（Excel 互換）

- セル参照：`=A1+B2`
- 範囲関数：`=SUM(A1:A5)`、`=AVERAGE(A1:A5)`、`=MAX(A1:A5)` など
- 四則演算・ネスト・複数セル参照
- 数式入力中に矢印キーでセル参照を挿入可能（Google Sheets 風）
  - `=` 入力後に矢印キーでセル参照が自動挿入される
  - 続けて矢印を押すと参照先が更新される
  - 演算子（+, -, *, / 等）を入力すると参照が確定し、次の矢印で新しい参照を開始
  - 参照中のセルは紫色の破線枠でハイライト表示

### Google Sheets ライクなキーボード操作

ProseMirror（BlockNote）がキーイベントを横取りするため、document capture フェーズでキーボードイベントをインターセプトして Google Sheets ライクな操作性を実現。

| 操作 | 動作 |
|---|---|
| 矢印キー | セル移動（Shift で範囲選択、Ctrl で端まで移動） |
| Enter | 編集モードに入る |
| Enter（編集中） | 確定して下のセルに移動 |
| F2 | 編集モードに入る |
| Escape（編集中） | 編集をキャンセル |
| Tab / Shift+Tab | 右 / 左に移動 |
| Delete / Backspace | 選択セルの内容をクリア |
| 文字入力 | セル内容を置き換えて編集開始 |
| Ctrl+Z / Ctrl+Y | Undo / Redo |
| Ctrl+C / Ctrl+X | コピー / カット |
| Ctrl+V | クリップボードからペースト |
| Ctrl+A | 全セル選択 |

### UI コンポーネント

**ツールバー（展開時）：**
- 最小化 / 展開ボタン
- CSV 読み込みボタン
- グラフ追加ボタン → ChartBlock を直下に挿入
- タイトル表示トグルボタン
- 行追加（+行）ボタン
- 列追加（+列）ボタン

**タイトル：**
- 「タイトル」ボタンで表示 / 非表示を切り替え
- 折りたたみ時はタイトルと行列数（例: "3 列 × 5 行"）を表示

**行・列の削除：**
- 行インデックスセル・列ヘッダーにホバーするとゴミ箱アイコンが表示
- クリックで該当行 / 列を削除

**セル選択の表示：**
- 現在セル：青色（#1a73e8）4辺ソリッド枠線、塗りつぶしなし
- 範囲選択：青色4辺ソリッド枠線 + 薄い青背景
- 行インデックス列：最小幅（32px）

### データ永続化

- テーブルデータは `lablate_dataset_<datasetId>` として localStorage に保存
- データ変更時に `CustomEvent("lablate-dataset-change")` を発火し、グラフブロックに通知

## グラフブロック（ChartBlock）

### 基本機能

- テーブルデータから XY 散布図を生成（Plotly.js）
- テーブルのデータ変更をリアルタイムに反映（CustomEvent による連携）
- 最小化 / 展開対応（折りたたみ時はタイトルまたは軸名を表示）

### 設定項目

| 設定 | 説明 |
|---|---|
| グラフタイトル | テキスト入力（空の場合は "Y vs X" 形式で自動生成） |
| X 軸列 | ドロップダウンで選択 |
| Y 軸列 | ドロップダウンで選択 |
| ヘッダー行 | 列ラベルとして使用する行（なし / 行番号） |
| データ行範囲 | 開始行〜終了行（空 = 全行） |

### 軸範囲の変更

- Plotly のビルトインのドラッグズームで軸範囲を変更可能
- プロット領域の四辺に薄いグレー（#d1d5db）の境界線を表示し、操作可能エリアを示す
- ホバー時にモードバー（ズーム / パン / リセット等）を表示

### リサイズ機能

- ブロックをクリックすると8つのリサイズハンドル（角4つ + 辺中央4つ）が表示
- ドラッグでブロック全体の幅・高さを変更
- ドラッグ中は青い破線枠とサイズ表示（"W × H"）のオーバーレイ
- サイズは `chartWidth`（0 = 自動）・`chartHeight`（デフォルト 340px）として設定に保存
- Plotly は `responsive: true` + `autosize: true` でコンテナに追従

### ブロック構造

```typescript
createReactBlockSpec({
  type: "chart",
  propSchema: { datasetId: { default: "" } },
  content: "none",
})
```

## データ保存（localStorage）

| キー | 内容 |
|---|---|
| `lablate_tree` | ページツリー |
| `lablate_doc_<pageId>` | BlockNote ドキュメント（ブロック配列） |
| `lablate_dataset_<datasetId>` | テーブルデータ（headers + rows）、ChartBlock と共有 |
| `lablate_chart_config_<blockId>` | グラフ設定（軸、タイトル、サイズ、折りたたみ等） |
| `lablate_collapsed_<blockId>` | テーブルブロックの折りたたみ状態 |
| `lablate_table_cfg_<blockId>` | テーブルタイトル設定（title, showTitle） |
